Prompts Collection is a curated repository of AI agents, prompts, and documentation generation tools designed for advanced AI-driven workflows. Implementation Guide

Using the Deep Research Agent

  1. Define your topic (select technical subject, audience level, focus areas)
  2. Run the agent with input parameters
  3. Review & iterate on the generated study guide

Using the Document Generator

  1. Prepare your project documentation
  2. Run the generator script
  3. Customize appearance (colors, layout)
  4. Deploy HTML file or upload to GitHub Pages

Technical Specifications

Supported Diagram Types

  • Mermaid Flowcharts (system architecture, workflows)
  • Class Diagrams (object-oriented relationships)
  • Sequence Diagrams (process interactions)
  • ER Diagrams (database schemas)
  • State Diagrams (state machines)
  • Gantt Charts (project timelines)

HTML Export Features

  • Sidebar Navigation: Fixed, smooth-scrolling TOC
  • Typography: Modern sans-serif with mono code fonts
  • Colors: Professional gradient headers, accessible contrast
  • Responsiveness: Mobile (320px), Tablet (768px), Desktop (1024px+)
  • Interactivity: Copy buttons, smooth navigation, section highlighting
  • Print: Clean layout without navigation elements
  • Accessibility: Semantic HTML, ARIA labels, keyboard navigation
