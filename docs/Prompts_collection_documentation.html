Debug Mode Setup

# Enable comprehensive logging for agents
import logging
import sys

# Configure logging
logging.basicConfig(
    level=logging.DEBUG,
    format='%(asctime)s - %(name)s - [%(levelname)s] - %(message)s',
    handlers=[
        logging.FileHandler('agent_debug.log'),
        logging.StreamHandler(sys.stdout)
    ]
)

logger = logging.getLogger('prompts_collection')

# Log agent interactions
def log_agent_query(agent_name, query, response):
    logger.debug(f"Agent: {agent_name}")
    logger.debug(f"Query length: {len(query)} chars")
    logger.debug(f"Query preview: {query[:200]}...")
    logger.debug(f"Response length: {len(response)} chars")
    logger.debug(f"Response preview: {response[:200]}...")

# Usage
log_agent_query("deep_research_agent", user_query, agent_response)

üìö Best Practices

Agent Usage Guidelines

‚úÖ DO:
  • Provide specific, detailed queries with context
  • Include 2-3 examples for complex tasks
  • Specify output format and length requirements
  • Test with diverse edge cases before production
  • Iterate 2-3 times to refine quality
  • Monitor token usage and costs
‚ùå DON'T:
  • Use vague queries like "optimize this"
  • Expect perfect first-time outputs
  • Share sensitive data in queries
  • Use wrong agent for the task
  • Skip validation and testing
  • Ignore model limitations
